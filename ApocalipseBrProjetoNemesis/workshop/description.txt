ApocalipseBR Projeto Nemesis for Build 42.

B42 port author: Hypothetic.
Original B41 ResidentEvilNemesis author and asset credit: Nemeth.

Includes:
- Nemesis outfit ported from the old ResidentEvilNemesis mod.
- MrX outfit ported from the old ResidentEvilNemesis mod.
- Server-side escalating chance system that can spawn Nemesis around players.
- Compatibility with manual admin summoning through createhorde2 using the Nemesis outfit.

Admin summon command:
/createhorde2 -x 10600 -y 9800 -z 0 -count 1 -outfit Nemesis -health 4.5

Visual MrX summon command:
/createhorde2 -x 10600 -y 9800 -z 0 -count 1 -outfit MrX -health 2.0
